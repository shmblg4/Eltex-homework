## Компиляция и установка

### 1. Компиляция модуля
Скомпилируйте модуль:
```sh
make
```

### 2. Загрузка модуля
```sh
sudo insmod task_monitor.ko
```

### 3. Проверка работы
Вывод списка активных процессов:
```sh
cat /proc/task_monitor
```

### 4. Выгрузка модуля
```sh
sudo rmmod task_monitor
```
